23.10.11 수업 정리(seq09)

ArrayList, LinkedList
- 순서가 있는 데이터의 집합
- 중복 가능
- 길이가 정해져 있지 않을 때 Good!
- 배열과 다르게 toString이 재정의(오버라이드) 되어있어 변수명을 그대로 출력해도 변수 주소가 가르키는 데이터를 프린트
- for each문 사용할 때 Object
- indexOf()는 첫 번째만 알려줌

HashSet
- 순서를 유지하지 않는 데이터의 집합
- 중복 불가능

Map
- 키와 값이 쌍으로 이루어진 데이터의 집합
- 맵핑시키기, 연결시키기
- toString이 오버라이딩 되어있음
- 정렬이 되어있음
- add 대신 put을 사용함

2) network 연결 - DBMS 연결
